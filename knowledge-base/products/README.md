# Products

Place product specification files here in Markdown format.

## Template

```markdown
# [Product Name]

## Overview
Brief product description and target market.

## Specifications

| Spec | Value |
|------|-------|
| Model | XXX |
| Power | XXXW |
| Dimensions | XXX x XXX x XXX mm |
| Weight | XXX kg |
| Input Voltage | XXXV |

## Business Info

| Item | Details |
|------|---------|
| MOQ | XXX pcs |
| Lead Time | XX days |
| Payment Terms | T/T, L/C |
| Certifications | CE, FCC, RoHS |
| Packaging | XXX |

## Target Markets
- Market A
- Market B
```

## Naming Convention
- `solar-portable-power-station-300w.md`
- `home-appliance-refrigerator-200l.md`
