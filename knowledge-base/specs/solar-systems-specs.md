## 5. Test Reports Required for B2B Whitlesale

Wholesale buyers typically request:
1. IEC 61215 + IEC 61730 test report (solar panels)
2. IEC 62109-1 safety report (inverters)
3. IEC 62619 + UN38.3 (batteries)
4. CE DoC and test report (all products)
5. Flash test report (solar panels, per batch)
6. On-site factory audit report (ISO 9001 preferred)
